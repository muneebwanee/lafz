# **App Name**: LAFZ

## Core Features:

- Word List: Browse the list of 600 core Quranic words.
- Visual Word Highlighting: View each word in visually distinct boxes or highlight to ensure easy readability.
- Theme Switching: Toggle between dark and light themes to suit user preference and reading environment.
- Multilingual Definitions: Access definitions and example usages of each word in English, Urdu, and Hinglish.
- Contextual Examples: The app uses an LLM 'tool' to provide relevant examples from the Quran for each word to facilitate understanding and contextual learning.
- Progress Tracking: Allow users to mark words as 'learned' for progress tracking.

## Style Guidelines:

- Primary color: A subdued purple (#A090D0) evokes a sense of spirituality and tranquility without being too somber.
- Background color: A very light gray (#F0F0F3) in light theme and a dark charcoal (#212429) in dark theme provides a comfortable contrast for reading.
- Accent color: A soft green (#80BCA3) highlights interactive elements, conveying growth and understanding.
- Font pairing: 'Literata' (serif) for body text to enhance readability in longer passages, and 'Space Grotesk' (sans-serif) for headlines to maintain a modern feel.
- Adopts a clean, card-based UI inspired by Instagram, promoting easy navigation and content discovery.
- Use clear, minimalist icons for navigation and actions to ensure ease of use. Select icons should work well in both light and dark themes.
- Implement subtle transitions and animations to provide feedback on user interactions.
